# 🛒 Blinkit Sales Analysis (Power BI)

This project presents an interactive **Blinkit Sales Dashboard** built using Power BI to analyze sales performance, outlet distribution, and customer trends across different categories.

## 🔍 Key Insights
- **Total Sales:** 1.20M+
- **Average Sales:** 140.99
- **Total Items Sold:** 9K+
- **Average Rating:** 3.92
- **Top Performing Outlet Type:** Supermarket Type1
- **Highest Sales Contribution:** Tier 3 locations
- **Most Sold Categories:** Fruits & Vegetables, Snack Foods
- Medium-sized outlets generate the highest sales

## 📈 Features
- Outlet-wise and location-based sales analysis
- Item category performance tracking
- Outlet size and establishment trend analysis
- Fat content-based sales comparison
- Customer rating and item visibility insights

## 🛠 Tools Used
- Power BI  
- Data Analysis  
- Data Visualization  

## 🎯 Purpose
The objective of this project is to understand sales patterns, outlet performance, and customer preferences to support better business decisions and demonstrate data analytics skills.
